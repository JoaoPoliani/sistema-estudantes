# \# Sistema de Cadastro de Estudantes

# 

# API REST simples desenvolvida com Spring Boot para consulta de cursos e estudantes.

# 

# \## Endpoints Disponíveis

# 

# \### Informações do Sistema

# GET /api/sistema/info

# 

# \### Listar Cursos

# GET /api/cursos

# 

# \### Buscar Curso por ID

# GET /api/cursos/{id}

# 

# \### Buscar Estudante por Matrícula

# GET /api/estudantes/{matricula}

# 

# \### Buscar Estudantes por Curso

# GET /api/estudantes/curso/{curso}

# 

# \## Como Executar

# 

# 1\. Clonar o projeto

# 2\. Abrir em uma IDE (IntelliJ ou Eclipse)

# 3\. Executar a classe SistemaEstudantesApplication

# 4\. Acessar via navegador ou Postman

# 

# \## Exemplos de Uso

# 

# \- http://localhost:8080/api/sistema/info

# \- http://localhost:8080/api/cursos

# \- http://localhost:8080/api/cursos/1

# \- http://localhost:8080/api/estudantes/2024001

# \- http://localhost:8080/api/estudantes/curso/Engenharia de Software
